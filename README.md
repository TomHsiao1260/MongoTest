## Introduction

<h4 align="justify">
這是個嘗試將記帳變得有趣的網站。用來記錄兩個使用者之間的小額借還款記錄，同時也具備聊天的功能。網頁中每個使用者各持有一種類型的角色，其數量會隨著兩位使用者之間借還款的總和而增加、減少。
<h4/>

<h4 align="justify">
此外，帳目和聊天資訊會儲存在 MongoDB 雲端資料庫，並透過 socket.io 對所有裝置做即時的同步更新。 
<h4/>

<h4>
使用技術： React、Socket、MongoDB、CSS Animation
<h4/>

## Installation
